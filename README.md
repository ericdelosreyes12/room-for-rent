# Room Rental Website

Static room rental website hosted on GitHub Pages.

## Features

- Responsive design
- Room gallery
- Virtual tour section
- Google Maps exact location
- Reservation via landlord contact
- GitHub Pages ready

## How to Deploy to GitHub Pages

1. Create a GitHub repository.
2. Upload all project files.
3. Go to:

   Settings → Pages

4. Under "Branch", select:

   main → /root

5. Click Save.

6. Your website will be available at:

```text
https://yourusername.github.io/repository-name/
